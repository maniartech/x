# calc.Asinh

Asinh returns inverse hperbolic sine of a number.

## Syntax

```go
Asinh(number)
```

## Arguments

### number

Required. The number is an input to Asinh function, it can be any real number.

## Example

```Go
Asinh(-10.1653) // Returns -3.014537762282636
Asinh(-1546)  // Returns -8.03657351430473
```
