# calc.CSC

CSC function returns cosecant of a number.

## Syntax

```go
CSC(number)
```

## Arguments

### number

Required. The number is an input to the CSC funtion, it can be any real number.

## Example

```Go
CSC(-36) // Returns  1.0082892940581898
CSC(0.235)  // Returns 4.294739640961497

```
