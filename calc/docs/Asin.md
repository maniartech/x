# calc.Asin

Asin returns inverse sine of a number.

## Syntax

```go
Asin(number)
```

## Arguments

### number

Required. The number is an input to Asin function, it must be between -1 and 1.

## Example

```Go
Asin(0.26) // Returns 0.2630222029084689
Asin(-0.546)  // Returns -0.5775822745289381
```
