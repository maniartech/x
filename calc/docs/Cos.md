# calc.Cos

Cos function returns the cosine of the given  number.

## Syntax

```go
Cos(number)
```

## Arguments

### number

Required. The numbers is the angle in radian for which you want cosine.

## Example

```Go
Cos(0.235) // Returns 0.9725143413326429
Cos(-3.5)  // Returns -0.9364566872907963
```
