# calc.Deg2Rad

Deg2Rad function converts degree to radian.

## Syntax

```go
Deg2Rad(angle)
```

## Arguments

### angle

Required. The angle is a degree that you want to convert to radian.

## Example

```Go
Deg2Rad(10) // Returns  0.17453292519943295
Deg2Rad(-10) // Returns -0.17453292519943295
```
