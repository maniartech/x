# calc.Rad2Deg

Rad2Deg function converts radian to degree.

## Syntax

```go
Rad2Deg(angle)
```

## Arguments

### angle

Required. The angle is in radian that you want to convert to degree.

## Example

```Go
Rad2Deg(-10.15631) // Returns -581.9136984265131
Rad2Deg(231.5163) // Returns 13264.90687848462

```
